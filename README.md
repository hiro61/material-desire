# Material Desire

Material Desire is a local-first PWA for observing purchase impulses, waiting before buying, and reevaluating decisions with calmer judgment.

## Features

- Quick desire capture with a circular intensity control
- Waitlist with due-time tracking
- Reevaluation prompts and recommendation flow
- Lightweight analytics for triggers and outcomes
- JSON export/import and full local reset
- Offline caching via service worker

## Local run

```bash
python3 -m http.server 4174
```

Open `http://127.0.0.1:4174/` in a browser.

## Deployment

This app is static and can be deployed directly to GitHub Pages from the repository root.
